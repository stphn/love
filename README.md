# LÖVE Games Collection

A collection of 2D games built with [LÖVE](https://love2d.org/) (Lua game framework) for learning game development and experimenting with Lua programming.

## About

This repository serves as a playground for learning Lua and game development with the LÖVE framework. Each game explores different mechanics, algorithms, and game design concepts.

## Games

### Snake
A classic snake game with multiple game modes and LCD-style graphics.
- Location: `snake/`
- See [snake/README.md](snake/README.md) for details

## Getting Started

### Prerequisites
- Install [LÖVE](https://love2d.org/) for your platform

### Running a Game
```bash
# Navigate to a game directory and run
cd snake
love src/
```
